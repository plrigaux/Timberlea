# Salt Pepper

A client server HTTP file manager that allows to upload and serve files.

## Features

* Upload file
* Download file
* Copy file
* Rename file and directory
* Delete file and directory
* Browse file and directory
* Create file
* Create directory

## Upcoming features

* Authentification
* Profile
  * File access per profile
  * Action per profile
* Filter files
* CLI client
* Zip download

## Installation

`npm i`

## Running

`npm run serve`
